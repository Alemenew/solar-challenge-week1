## Setup

1. Create a virtual environment:
```bash
python -m venv .venv
source .venv/bin/activate  # On Windows: .venv\Scripts\activate
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

## Project Structure

- `src/` - Source code
- `notebooks/` - Jupyter notebooks
- `tests/` - Unit tests
- `scripts/` - Utility scripts

## Running Tests

```bash
pytest tests/ -v
```
